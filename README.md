# 📨 PQRS-System (Python & React Version)

### Gestión Avanzada de Peticiones, Quejas, Reclamos y Sugerencias

Este es un sistema moderno desarrollado por **Paramo Labs** que utiliza un stack robusto para ofrecer una experiencia de usuario fluida y un backend escalable. 

---

## 🛠️ Stack Tecnológico

* **Backend:** ![Python](https://img.shields.io/badge/Python-3776AB?style=flat&logo=python&logoColor=white) (FastAPI/Flask - el que uses)
* **Frontend:** ![React](https://img.shields.io/badge/React-20232A?style=flat&logo=react&logoColor=61DAFB) ![Vite](https://img.shields.io/badge/Vite-646CFF?style=flat&logo=vite&logoColor=white)
* **Base de Datos:** SQLite / PostgreSQL (vía SQLAlchemy)
* **Gestión de Proyectos:** Trello (Fase 1 completada)

---

## 🚀 Características
- Interfaz reactiva con React.
- API RESTful en Python.
- Gestión de estados de solicitudes en tiempo real.
- Documentación de procesos incluida en la carpeta `uploads`.
